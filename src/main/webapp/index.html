JavaJEE.com - Deploying a Bottoms-Up Web Service in Tomcat

This project supports a personal walk through of the JavaJee.com Services and XML tutorial.

In this Web application / WAR example, we show the results of the tutorial section titled Deploying a Bottoms-Up Web Service in Tomcat. In our case, we resolve dependencies using Gradle and may run the example in Tomcat, Jetty, or any supported Servlet container.
